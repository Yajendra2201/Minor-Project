As the name suggests E-Cropter targets the information related to the crop with the help of Machine Learning and provides appropriate information regarding crop production at a particular location/area. E-Cropter is a web application that predicts weather with the help of dataset/API. Firstly, it takes the location from user after providing authentication to access the website. It is a multi-disciplinary application that gives information by performing mathematical logics and algorithm to determine which crop can be grown as per weather conditions on the location provide by user. So, the main motive of this project is to enhance the production of crop for the user by crop prediction.
